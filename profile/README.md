# AllaTre

AllaTre is a **real-time online auction platform** built to support structured, transparent, and competitive C2C auctions, with a strong focus on the Egyptian market.

---

## Overview

AllaTre enables users to create and participate in timed auctions with live bidding, reserve prices, and instant updates. The platform solves the limitations of traditional classified marketplaces by providing a complete auction lifecycle with real-time interaction.

---

## Key Features

* Timed auctions with automatic state management
* Real-time bidding using WebSockets
* Reserve price & Buy It Now support
* Secure bid validation and winner determination
* Ratings, reviews, reporting, and admin moderation
* Real-time notifications (outbid, auction ending, auction won)

---

## AI Features

* **Semantic Search** for accurate and relevant auction discovery
* **Recommendation System** based on user interactions
* **Policy-aware Chatbot (RAG)** for user assistance

---

## Tech Stack

* **Backend:** Node.js, Express, FastAPI
* **Frontend:** React Native (mobile), React (admin dashboard)
* **Database:** PostgreSQL
* **Real-time:** WebSockets
* **Search & AI:** Meilisearch, FAISS, embedding models

---

## Architecture

* Modular monolithic backend
* Separate AI services
* Secure REST APIs and WebSocket connections
